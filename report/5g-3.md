# Мат модель
$P_\text{loss} = 1 - P_\text{visibility}$

$P_\text{visibility} = P_\text{sectors} * P_\text{nonblock}$

$P_\text{sectors} = P_\text{sector}^2$

$P_\text{sector} = \frac{fov}{2\pi}$

$P_\text{nonblock} = e^{-\lambda S}$

$S = 2rh$
